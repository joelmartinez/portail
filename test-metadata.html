How It Works

  • Static Metadata: Add data-metadata to any element with a JSON object
  • Dynamic Metadata: Add custom data-* attributes to interactive elements
  • Automatic Collection: On interaction, all data attributes are collected
  • AI Integration: Metadata is formatted and passed to the AI prompt
  • Continuity: The AI uses metadata to create contextual, personalized experiences

Testing in Portail

To test this feature in the actual Portail application:

  1. Open the Portail application
  2. Ask the AI to create an experience with metadata (e.g., "Create an RPG with stat tracking")
  3. The AI can now use the metadata support to add data-metadata and data-* attributes
  4. When you interact with elements, metadata is automatically collected and passed to the next experience
  5. The AI receives this metadata and can use it to maintain continuity and context
